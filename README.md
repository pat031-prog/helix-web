# helix-web

Frontend-only repo for the Helix Proto interface.

This repository contains:

- landing page
- app shell
- streaming chat / agent / knowledge UI
- runtime backend URL connector
- Vercel routing config

## Deploy on Vercel

Use the repo root as the project root.

- Framework Preset: `Other`
- Build Command: leave empty
- Output Directory: leave empty

After deploy, open `/app`, paste the backend URL, and click `Connect`.
